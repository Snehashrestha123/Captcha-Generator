# Captcha Generator

A simple CAPTCHA generator that creates random codes and verifies user input in real time.

![Screenshot](./screenshot.png)

## About

This project is a web-based CAPTCHA generator built using HTML, CSS, and JavaScript.  
It generates random CAPTCHA text, displays it to the user, and checks whether the entered value matches the generated code.  
I built this project to practice random string generation, DOM manipulation, and form validation using JavaScript.

## Built With

`HTML` `CSS` `JavaScript`

## What I Learned

- How to generate random values and validate user input dynamically using JavaScript.
